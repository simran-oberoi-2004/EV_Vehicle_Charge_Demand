# ⚡ EV Vehicle Charge Demand Forecasting

## 📌 Overview

This project focuses on **Electric Vehicle (EV) adoption and charging demand forecasting** using real-world data. It leverages **machine learning models and data analysis techniques** to predict future EV growth trends, enabling better planning for **charging infrastructure and energy demand management**.

The application is deployed using **Streamlit**, providing an interactive interface for users to explore predictions and insights.

---

## ✨ Features

* 📊 **EV Adoption Forecasting** – Predict future electric vehicle growth trends
* ⚡ **Charging Demand Estimation** – Analyze potential energy and infrastructure needs
* 🤖 **Machine Learning Model** – Pre-trained model for accurate predictions
* 📈 **Data Visualization** – Interactive charts and insights
* 🌐 **Streamlit Web App** – User-friendly interface for real-time interaction
* 📂 **Preprocessed Dataset** – Cleaned and structured data for efficient analysis

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn, Matplotlib/Seaborn
* **Framework:** Streamlit
* **Model:** Regression-based forecasting model (`.pkl`)
* **Environment:** Jupyter Notebook for development

---

## 📂 Project Structure

* `app.py` – Streamlit application for interactive predictions
* `EV_Adoption_Forecasting.ipynb` – Model development and data analysis
* `forecasting_ev_model.pkl` – Trained machine learning model
* `preprocessed_ev_data.csv` – Cleaned dataset used for training
* `Electric_Vehicle_Population_By_County.csv` – Raw dataset
* `.streamlit/config.toml` – Streamlit configuration
* `requirements.txt` – Project dependencies
* `runtime.txt` – Deployment environment configuration
* `ev-car-factory.jpg` – UI/visual asset

---

## 🚀 How It Works

1. Data is collected and preprocessed from EV population datasets
2. Feature engineering is applied to extract meaningful patterns
3. A **machine learning regression model** is trained on historical data
4. The trained model predicts future EV adoption trends
5. Streamlit app displays results through **interactive visualizations**
